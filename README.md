# Mini-Hackathon-
Presented "TravelWise: Smart Travel Planning Platform" in 'Mini-Hackathon' conducted in ACS College. This is a platform where users can find their desired destination, convenient mode of transport and other facilities available nearby and can compare their trip expenses in different modes. User can also raise a query is they have any of such.
